# DiRAC HPC Benchmarking Ressources

This repository collects benchmarking information for various DiRAC HPC architectures and codes, as well as information and tutorials for benchmarking.  

##  Benchmarking Tools
A directory for information and tutorials of benchmarking tools on the various DiRAC HPC systems.

## Architectures
A directory for architecture information and expected peak performance of various DiRAC HPC systems.

##  Codes
A directory for benchmarking logs of DiRAC HPC codes.
